# Weather App

This **Weather App** provides real-time weather updates for any city entered by the user. It fetches weather data from the OpenWeatherMap API and displays details like temperature, humidity, wind speed, and weather conditions with appropriate icons. The app features a simple, responsive UI for ease of use.

## Features
- **City Search**: Enter any city name to get current weather data.
- **Weather Details**: Displays temperature, humidity, wind speed, and dynamic weather icons.
- **Error Handling**: Shows an error message if the city name is invalid or not found.

## Technologies Used
- **HTML5**: For structuring the web page.
- **CSS3**: For styling the app's user interface.
- **JavaScript (ES6)**: For making asynchronous API requests and dynamically updating the DOM.
- **OpenWeatherMap API**: Provides real-time weather data.
